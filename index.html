Politica de privacidade para minhas watchfaces

Eu Rafaela, construí essas watchfaces para WearOS.

Esta página é usada para informar os visitantes sobre minhas políticas com a coleção, uso e divulgação de informações pessoais, se alguém decidiu usar meu serviço.

Coleta de informações e uso

Não coleto conscientemente nenhuma informação dos usuários de nenhuma de minhas watchfaces, todos os dados mostrados são lidos de outros aplicativos e não possuo acesso a eles.

Coleta de dados de saúde

Não coleto conscientemente nenhum dado de saúde ou condicionamento físico dos usuários de nenhuma de minhas watchfaces, todos os dados mostrados são lidos de outros aplicativos e não possuo acesso a eles.

Privacidade das crianças

Não coleciono conscientemente nenhuma informação de crianças menores de 13 anos.

Mudanças nesta Política de Privacidade

Posso atualizar nossa política de privacidade de tempos em tempos. Assim, você é aconselhado a revisar esta página periodicamente para quaisquer alterações. Vou notificá-lo sobre quaisquer alterações publicando a nova política de privacidade nesta página. Esta política é eficaz a partir de 2025-10-26
